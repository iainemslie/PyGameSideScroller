# PyGameSideScroller

![alt text](image.png)

## Create the environment

### macOS/Linux

```sh
$ python3 -m venv gameenv
$ source gameenv/bin/activate
$ pip install requirements.txt
```

### Windows

```sh
$ python -m venv gameenv
$ gameenv\Scripts\activate
$ pip install requirements.txt
```

## Run the game

### macOS/Linux

```sh
$ python3 code/main.py
```

### Windows

```sh
$ python code\main.py
```
